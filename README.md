# Demo site for 'AI in the Screen Sector: Casebook'
This repository was created as a demo showcase for an online collection of case studies, aligned with the CoSTAR Foresight Lab's work tracking adoption of generative AI throughout the screen sector.

The case studies are now published at [costar-foresight-lab.github.io/ai-casebook/](https://costar-foresight-lab.github.io/ai-casebook/). The active GitHub repository is at [costar-foresight-lab/ai-casebook](https://github.com/costar-foresight-lab/ai-casebook).
